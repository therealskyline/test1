Anime Zone Documentation

Introduction

Welcome to the Anime Zone documentation! This guide explains how to add new anime, seasons, and episodes to the platform. The site uses a JSON-based data storage system, making it easy to manage content without a complex database.

Data Structure

All anime data is stored in static/data/anime.json. This file contains an array of anime objects, each with their own properties and nested arrays for seasons and episodes.

Anime Object Structure

{
  "id": 1,                          // Unique identifier for the anime
  "title": "Anime Title",           // The title of the anime
  "description": "Description...",  // A brief description of the anime
  "image": "image_url",             // URL to the anime poster image
  "genres": ["action", "fantasy"],  // Array of genres
  "rating": 8.5,                    // Rating out of 10
  "featured": true,                 // Whether to show on homepage
  "seasons": []                     // Array of seasons (see below)
}

Season Object Structure

{
  "season_number": 1,               // Season number
  "episodes": []                    // Array of episodes (see below)
}

Episode Object Structure

{
  "episode_number": 1,              // Episode number within the season
  "title": "Episode Title",         // Title of the episode
  "description": "Description...",  // Brief description of the episode
  "video_url": "video_url"          // URL to the video file
}

How to Add New Content

Adding a New Anime

To add a new anime, you need to add a new object to the anime array in the JSON file. Make sure to give it a unique ID and include all the required fields. 

// Example of adding a new anime
{
  "id": 21,  // Make sure this is unique
  "title": "New Anime Title",
  "description": "This is a description of the new anime.",
  "image": "https://example.com/image.jpg",
  "genres": ["action", "comedy"],
  "rating": 8.7,
  "featured": false,
  "seasons": [
    {
      "season_number": 1,
      "episodes": [
        {
          "episode_number": 1,
          "title": "First Episode",
          "description": "This is the first episode.",
          "video_url": "https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing"
        }
      ]
    }
  ]
}

Adding a New Season to an Existing Anime

To add a new season, find the anime in the JSON file and add a new object to its seasons array. 

// Find the anime by its ID and add a new season
{
  "season_number": 2,  // Make sure this is the next season number
  "episodes": [
    {
      "episode_number": 1,
      "title": "First Episode of Season 2",
      "description": "This is the first episode of season 2.",
      "video_url": "https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing"
    }
  ]
}

Adding a New Episode to an Existing Season

To add a new episode, find the anime and season in the JSON file, and add a new object to the season's episodes array. 

// Find the anime and season, then add a new episode
{
  "episode_number": 2,  // Make sure this is the next episode number in the season
  "title": "Second Episode",
  "description": "This is the second episode of the season.",
  "video_url": "https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing"
}

Best Practices

  • Always use unique IDs for anime entries
  • Keep descriptions concise but informative
  • Use high-quality images for anime posters
  • Make sure video URLs point to valid and playable videos
  • Backup the JSON file before making significant changes
  • Format genres consistently (lowercase, no spaces)
  • Set "featured: true" only for anime you want to highlight on the homepage

Example: Complete Workflow

Here's a complete example of how to add a new anime with multiple seasons and episodes:

Step 1: Create the Anime Object

{
  "id": 21,
  "title": "Dragon Hunter",
  "description": "A young warrior embarks on a journey to hunt dragons and save his village.",
  "image": "https://example.com/dragon-hunter.jpg",
  "genres": ["action", "fantasy", "adventure"],
  "rating": 8.9,
  "featured": true,
  "seasons": []
}

Step 2: Add the First Season

"seasons": [
  {
    "season_number": 1,
    "episodes": []
  }
]

Step 3: Add Episodes to the Season

"episodes": [
  {
    "episode_number": 1,
    "title": "The Beginning",
    "description": "Ryu discovers his destiny as a dragon hunter when his village is attacked.",
    "video_url": "https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing"
  },
  {
    "episode_number": 2,
    "title": "Training Begins",
    "description": "Ryu starts his training under a veteran dragon hunter.",
    "video_url": "https://drive.google.com/file/d/YOUR_FILE_ID/view?usp=sharing"
  }
]

Step 4: Add the Complete Object to the JSON File

Finally, add the complete anime object to the anime array in the JSON file. Make sure to place it inside the array and use proper JSON formatting with commas between objects.
